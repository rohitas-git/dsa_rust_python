
We have collection of items on which we have to do
1. Search
2. Insert
3. Delete
4. Find closest 
5. Sorted Traversal

Array (Unsorted)
1. O(n)
2. O(1)
3. O(n)
4. O(n)
5. O(nlogn)

Array (Sorted)
1. O(logn)
2. O(n)
3. O(n)
4. O(logn)
5. O(n)

Linked List
1. O(n)
2. O(1) [O(n) in sorted]
3. O(n)
4. O(n)
5. O(nlogn) [O(n) in sorted]

BST (Balanced)
1. O(logn)
2. O(logn)
3. O(logn)
4. O(logn)
5. O(n)

Hash
1. O(1)
2. O(1)
3. O(1)
4. O(n)
5. O(nlogn)
